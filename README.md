# Sponge-and-rag-holder
A sponge and rag holder which can drain out the extra water


I desgined this to store the sponge and rag instead of just throwing it on the kitchen table, This way it is kept neatly, drains extra water and saves space from teh kitchen table,

To assemble it, You just need to place the rag holder on top of the sponge tray holder.
